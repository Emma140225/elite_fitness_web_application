# Elite Fitness - Web-Application

## Description

I developed this project as part of my retraining program. The gym doesn't actually exist, and all database entries are AI-generated. I created the web application on my own; the database was developed as part of a group project. The entire project is in German.

To use the application, you must first create the database in MySQL.

1. ef2_ddl.sql
2. ef2_constraints.dql
3.ef2_stammdaten.sql
4.ef2_bewegungsdaten.sql

I've always used XAMPP when developing the application; it lets me host the web server and MySQL server at the same time, and I can access everything via localhost.

## Content

The web application is a CRUD (Create/Read/Update/Delete) application. It allows users to manage members, trainers and their qualifications, members' RFID wristbands, contract types, and courses. There are also pages for statistics on courses, equipment, and members.

Not everything is finalized yet, and the statistics are mostly placeholders at this point. The project is at the stage it was in at the time of submission.
